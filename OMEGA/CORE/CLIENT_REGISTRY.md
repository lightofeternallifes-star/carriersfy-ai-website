## Client Onboarding Process

When a new client is signed:
1. Create a folder under `/OMEGA/CLIENTS/[ClientName]/`
2. Fill in all 9 standard client files (see template in [Future Clients](../CLIENTS/Future_Clients/Onboarding_Template.md))
3. Create a project entry under `/OMEGA/PROJECTS/[ClientName]/`
4. Add an entry to this registry
5. Assign AI employees as scoped
6. Create initial tasks in [TASK_ENGINE](TASK_ENGINE.md)
7. Schedule kickoff meeting via Sofia
