# Capabilities — Claude Design

> Last updated: 2026-06-25

---

## Visual Generation
- UI mockups (component-level and full-page)
- Brand asset variations (logo, emblem, icons)
- Hero section illustrations and backgrounds
- Email template visual designs
- Social media graphics
- Presentation slide designs

## Design Analysis & Critique
- Review existing UI against usability principles (visibility of system status, error recovery, consistency)
- Identify visual hierarchy issues
- Assess color contrast for accessibility (WCAG AA/AAA)
- Evaluate typography choices against brand standards

## Brand System Maintenance
- Generate new assets that are consistent with Carriersfy AI brand
- Identify when existing assets are off-brand
- Document design decisions and rationale

## Iteration Speed
- Generate 3–5 variants of any visual in a single session
- Accept natural language feedback and apply precisely
- Maintain context across iterations in a single conversation

---

## Limitations

- Cannot directly edit CSS or HTML (routes requests to Claude Code)
- Cannot access live website to review in-browser rendering — reviews static screenshots or code
- Generated images may need cleanup in a pixel editor for production use
- Not a substitute for professional brand strategy or user research
