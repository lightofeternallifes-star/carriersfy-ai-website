> Project README | Status: [ACTIVE — PIPELINE] | Last updated: 2026-06-25

## Overview

This project tracks the full pipeline from prospect identification through to signed client — and serves as the master playbook for how Carriersfy AI acquires, onboards, and delivers for new clients.

---

## Pipeline Stages

| Stage | Description | Owner |
|---|---|---|
| Prospect | Identified target — no contact yet | Iron Prime |
| Contacted | First outreach sent | Iron Prime |
| Qualified | Discovery call completed — real need confirmed | Juan + Iron Prime |
| Proposal Sent | Formal AI services proposal delivered | Juan |
| Negotiation | Terms being discussed | Juan |
| Signed | Contract executed | Juan |
| Onboarding | Sofia-managed kickoff process | Sofia |
| Active Client | Live in [CLIENT_REGISTRY](../../CORE/CLIENT_REGISTRY.md) | Juan + AI Workforce |
