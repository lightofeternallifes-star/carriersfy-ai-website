## Critical File: `functions/api/contact.js`

This is the only server-side file in the project. Key behavior:

```
POST /api/contact
→ Validates Content-Type: application/json
→ Parses JSON body
→ Requires: name (string), email (valid email)
→ Optional: business, phone, industry, message
→ Reads env.RESEND_API_KEY
→ POSTs to https://api.resend.com/emails
→ FROM: leads@carriersfy.ai
→ TO: juan@carriersfy.ai, hello@carriersfy.ai
→ Returns: { success: true } or { error: "..." }
```

**If `RESEND_API_KEY` is missing:** Returns 500 `{ error: "Server misconfiguration — contact admin" }`
